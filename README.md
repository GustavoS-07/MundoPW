# Gustavo Soares Pimentel 

## Gerenciador de Países e Cidades

### Feito utilizando as tecnologias: 
- SQL
- HTML
- PHP
- JavaScript
- CSS

### Descrição do Projeto

Sistema de centralização e gerenciamento de registros de países e cidades, completo com um banco de dados relacional, CRUD e integração das APIs Restcountries, para informações de capital, moeda, e bandeira dos países cadastrados, e Openweather para exibição de clima, temperatura e umidade de cidades cadastradas.
